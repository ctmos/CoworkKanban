System-Uebersicht

CoworkOS ist ein persoenliches Drei-Agenten-System fuer den klinischen Alltag. Es kombiniert ein visuelles Dashboard (LifeOS), einen 24/7-Chatbot (Hermine) und einen Development-Partner (KITT) zu einem integrierten Arbeitssystem. Gebaut mit Vanilla JS, gehostet auf GitHub Pages, synchronisiert ueber die GitHub API — ohne eigenen Backend-Server, ohne Framework, ohne Build-System.

Patientendaten werden clientseitig mit AES-256-GCM verschluesselt bevor sie GitHub erreichen. Alle sensiblen Repos sind privat, die Git-History ist bereinigt, und der Zugriff laeuft ueber einen Fine-Grained PAT mit minimalen Rechten. Der Token wird im Browser mit dem PIN verschluesselt gespeichert (PBKDF2-Vault) — kein Klartext in localStorage.

Das System pollt alle 60 Sekunden mit ETag-Conditional GET (304 Not Modified spart API-Kontingent), nutzt eine Write-Queue fuer konfliktfreie Schreibzugriffe und merged Patientendaten deterministisch auf Feldebene (updatedAt). Aktuell: SW v5.56, HTTPS enforced, CSP aktiv.
